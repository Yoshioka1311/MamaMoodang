## Quick Start On Windows

You do not need to type commands every time anymore.

1. Double-click `open-project.bat` to open this folder in VS Code.
2. Double-click `start-website.bat` to start the website.
3. Double-click `stop-website.bat` to stop the website.

Notes:

- This project uses Flutter, not `npm run dev`.
- This repo already works with Flutter from `D:\flutter\bin\flutter.bat`.
- If you move Flutter later, either update the script or add `flutter\bin` to your Windows `PATH`.
- The website will open at `http://127.0.0.1:3000`.
